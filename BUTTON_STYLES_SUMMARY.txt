🚀 QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Include CSS di <head>
────────────────────────────────────
<link rel="stylesheet" href="../css/buttons.css">
(Sesuaikan path tergantung lokasi file)


STEP 2: Gunakan Semantic Classes
────────────────────────────────────
<!-- Primary Button (Simpan) -->
<button type="submit" class="btn-primary">Simpan</button>

<!-- Cancel Button -->
<button class="btn-cancel">Batal</button>

<!-- Delete Button (Tabel) -->
<a href="?delete=1" class="delete btn-sm">Hapus</a>


STEP 3: Untuk Multiple Buttons
────────────────────────────────────
<div class="btn-group">
    <button class="btn-primary">Simpan</button>
    <button class="btn-cancel">Batal</button>
</div>


STEP 4: Test di Browser
────────────────────────────────────
✓ Desktop: Chrome, Firefox, Safari, Edge
✓ Mobile: iPhone (Safari), Android (Chrome)
✓ Check: Hover effects, responsive layout
