Primary Button (Biru)

Gunakan untuk aksi utama seperti Simpan, Submit, Tambah

Contoh Penggunaan:

Link Button
<button type="submit" class="btn-primary">Simpan</button> <input type="submit" value="Tambah">

Success Button (Hijau)

Gunakan untuk aksi positif seperti Konfirmasi, Setuju

Contoh Penggunaan:

<button class="btn-success">Konfirmasi</button> <input type="submit" class="success" value="Setuju">

Danger Button (Merah)

Gunakan untuk aksi berbahaya seperti Hapus

Contoh Penggunaan:

Hapus Data
<button class="btn-danger">Hapus</button> <a href="#" class="delete">Hapus Data</a>

Warning Button (Kuning)

Gunakan untuk aksi yang memerlukan perhatian khusus

Contoh Penggunaan:

<button class="btn-warning">Peringatan</button>

Info Button (Cyan)

Gunakan untuk aksi informasi atau lihat detail

Contoh Penggunaan:

Lihat Detail
<button class="btn-info">Informasi</button>

Secondary/Cancel Button (Abu-abu)

Gunakan untuk membatalkan atau aksi alternatif

Contoh Penggunaan:

Kembali
<button class="btn-cancel">Batal</button>

Edit & View Buttons

Gunakan untuk aksi edit dan lihat detail dalam tabel

Contoh Penggunaan:

Edit Lihat Hapus
<a href="?action=edit&id=1" class="edit btn-sm">Edit</a> <a href="detail.php?id=1" class="view btn-sm">Lihat</a>
